# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Standards

Expected behavior:

- Be respectful and constructive.
- Keep technical disagreement focused on evidence, tradeoffs, and project goals.
- Assume good faith while still asking for clear reasoning and tests.
- Respect maintainers' decisions about project scope and release readiness.

Unacceptable behavior:

- Harassment, threats, insults, or discriminatory language.
- Public or private abuse of contributors or maintainers.
- Deliberate disruption of project discussions or review processes.
- Publishing private information without explicit permission.

## Enforcement

Maintainers may remove comments, close issues or pull requests, restrict
participation, or ban contributors whose behavior is harmful to the project or
community.

Report conduct concerns to the project maintainers through the repository's
private contact channel. Security issues should follow [SECURITY.md](SECURITY.md);
avoid posting sensitive personal reports or vulnerabilities in public issues.
